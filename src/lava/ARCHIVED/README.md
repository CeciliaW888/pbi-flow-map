# Archived Code

This folder contains legacy code that is no longer used in the current version of the visual.

## bingmap-legacy/

Original Bing Maps integration from v1.x versions.

**Status:** DEPRECATED (Bing Maps discontinued by Microsoft)
**Replaced by:** `/src/lava/maplibre/` (MapLibre GL JS)
**Date Archived:** February 2026
**Last Version:** v1.4.9

This code is kept for reference purposes only and is not compiled into the current visual.

### Why It's Here

- Historical reference
- Understanding migration decisions
- Comparing old vs new implementation

### Can It Be Deleted?

Yes, but we keep it temporarily for:
- Rollback capability (if needed)
- Documentation of what changed
- Learning from the old implementation

It may be permanently deleted in v3.0.0+
